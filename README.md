# Aegis Bank - Site Web Mobile-First

## Description
Aegis Bank est une néobanque éthique et transparente, conçue pour une expérience utilisateur optimale sur mobile. Ce projet présente une interface simple et élégante, mettant en avant les valeurs de la banque avec un design responsive et des animations fluides.

## Fonctionnalités
- **Menu burger** : Navigation latérale pour mobiles.
- **Boutons de connexion** : Liens "Se connecter" et "S'inscrire" avec effets hover.
- **Contenu principal** : Titre, sous-titre, CTA pour télécharger l'app, et offre promotionnelle.
- **Réseaux sociaux** : Liens vers Telegram, Discord et Mastodon.
- **Footer compact** : Mentions légales et copyright.
- **Animations** : Transitions et effets visuels pour une UX engageante.
- **Mobile-first** : Optimisé pour mobiles, sans scroll vertical.

## Technologies utilisées
- **HTML5** : Structure sémantique avec header, main, footer.
- **CSS3** : Flexbox, animations, variables CSS, responsive avec clamp().
- **Code pur** : HTML et CSS uniquement, sans librairies.
